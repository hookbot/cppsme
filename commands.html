CPPS.me Commands

This is a list of all the commands we have available for our players. These can range from adding clothing items, to changing your nameglow, to checking a server's response time.

Player Commands

  • !AI *ID* - Replace *ID* with the ID of the item you want to add.
  • !JR *ID* - Replace *ID* with the ID of the room you would like to join.
  • !AF *ID* - Replace *ID* with the ID of the furniture you want to add.
  • !AS *ID* - Replace *ID* with the ID of the stamp you want to add.
  • !AC 5000 - Adds 5000 coins to your user.
  • !PIN *ID* - Allows you to use any item as pin, replace *ID* with the item's ID.
  • !UP *INITIALS* - Put the initials of a 'mascot' - eg: RH (Rockhopper), AA (Aunt Arctic), etc, or try combinations such as !UP Rockhopper, as some of them are coded for ease of use!
  • !NG *COLOR* *GLOW* - Sets your nameglow, replace *COLOR* and *GLOW* with a color hex code (codes available here)
  • !BC *TEXT* *COLOR* - Sets your bubblecolor, replace *TEXT* and *COLOR* with a color hex code (codes available here)
  • !ROOMID - Shows the ID of your current room.

Server Commands

These commands are for advanced users who are curious about server stats and information.

  • !PING - This command was made to check if you're still connected to the server.
  • !PONG - This command was made to check if you're still connected to the server.
  • !UPTIME - Shows the server's uptime.
  • !COUNT - Shows the amount of online users.
